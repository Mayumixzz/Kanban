🚀 Kanban Board

Aplicação de Kanban desenvolvida com Python e HTML/CSS para organização visual de tarefas em fluxo estruturado.

O projeto simula um quadro Kanban clássico com colunas de progresso e persistência automática dos dados no navegador.

📌 Funcionalidades

Criação de cartões

Movimentação entre colunas

Edição e exclusão de tarefas

Criação dinâmica de novas colunas

Exportação de dados em JSON

Importação de dados em JSON

Persistência automática no navegador (Local Storage)

🛠 Tecnologias utilizadas

Python

HTML5

CSS3

JavaScript

Git

GitHub

▶️ Como executar

Clone o repositório:

git clone https://github.com/Mayumixzz/Kanban.git

Acesse a pasta do projeto:

cd Kanban

Execute o arquivo Python para gerar o HTML:

python kanban.py

Abra o arquivo kanban.html no navegador.
